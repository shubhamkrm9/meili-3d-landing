# 🌍 Meili – Immersive 3D Travel Landing Page

An immersive storytelling-based 3D landing page built using modern frontend technologies like Next.js, Framer Motion, and React Three Fiber. This project demonstrates advanced animation techniques, parallax scrolling, and interactive user experience design.

## ✨ Features

- 🌌 Immersive storytelling-based landing experience  
- 🪐 Interactive 3D planet animation using image sequences  
- 🎬 Smooth scroll-based animations powered by Framer Motion  
- 🔁 Parallax scrolling with variable speeds  
- ⌨️ Typewriter text animation effect  
- 💡 Responsive design for desktop and mobile  
- 🎯 Modern UI with hover effects and smooth transitions  
- ♿ Accessibility support (prefers-reduced-motion)  
- ⚡ Optimized performance and smooth rendering

## 📸 Project Overview

This landing page is designed to simulate a futuristic travel experience using immersive storytelling and animation.

The experience consists of multiple frames:

1. Travel introduction section
2. Customer interaction section
3. Real-time plan editing simulation
4. Travel agent dashboard view
5. Crisis handling simulation with animated planet
6. Call-to-action booking section

Each section uses coordinated animations and parallax effects to create a seamless storytelling flow.
